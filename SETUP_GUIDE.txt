STEP 1 — Upload static site to Netlify Drop (already works)
──────────────────────────────────────
1. Go to: https://netlify.com/drop
2. Drag the eid-firebase FOLDER (not zip)
   — drag all files EXCEPT GOOGLE_APPS_SCRIPT.js and SETUP_GUIDE.txt
   — drag: index.html, manifest.json, sw.js, firebase-messaging-sw.js,
            icon.svg, icon-192.png, icon-512.png
3. Done — site is live at eidskintracker.netlify.app ✅

STEP 2 — Set up Google Apps Script (FREE, runs in browser)
──────────────────────────────────────
1. Go to: https://script.google.com
   (sign in with any Google account)
2. Click "New project"
3. DELETE all existing code in the editor
4. Open GOOGLE_APPS_SCRIPT.js from this folder
5. Copy the ENTIRE contents → paste into the editor
6. Click the floppy disk icon (Save) — name it: Eid Skin Reminders
7. Click "Run" → select "testConnection" → click Run
   → It will ask for permissions → click Allow
   → Check "Execution log" at bottom — should show ✅ Everything working!

STEP 3 — Set the 15-minute trigger
──────────────────────────────────────
1. In Google Apps Script — click the clock icon (Triggers) on left sidebar
2. Click "+ Add Trigger" (bottom right)
3. Set these options:
   • Function to run: sendReminders
   • Event source: Time-driven
   • Type: Minutes timer
   • Interval: Every 15 minutes
4. Click Save
5. Done! Google will run sendReminders every 15 minutes for FREE ✅

STEP 4 — Set up on your phone
──────────────────────────────────────
1. Open https://eidskintracker.netlify.app in Chrome
2. Tap ⋮ menu → Add to Home Screen → install
3. Open app from home screen icon
4. Settings → tap "Allow Notifications"
5. Set your Sehri, Iftar, Night times
6. Tap "Save & Schedule Reminders" ✅

  HOW IT WORKS (no server needed):
  
  Your phone → registers with Firebase → gets FCM token
  Token + reminder times → saved to Firebase DB
  Google Apps Script → checks DB every 15 minutes (FREE)
  If it's Sehri/Iftar/Night time → sends FCM push
  Google FCM → delivers to your phone even when locked ✅

  ✅ Netlify Drop — only for static files (works fine)
  ✅ Google Apps Script — handles the scheduling (free)
  ✅ Firebase DB — stores your token + times (free)
  ✅ FCM — delivers the push notification (free)
  ✅ Total cost: $0. No credit card. No CLI. No install.
